# Team Defense Calls and Signals

## Summary
To change defenses fluidly during a game, every team needs a simple communication system so players can react instantly without confusion. Herb Brown's numbering system uses two-digit codes: the tens digit identifies the **type** of defense and the ones digit identifies **where** on the court pressure begins. Supplementary verbal cues ("hard," "soft," "marry/deny") add nuance without complexity. [S1, pp.179-180]

## When to Use
- Calling a defense from the sideline during a dead ball or timeout
- Communicating mid-game defensive changes between possessions
- Building a team vocabulary so defensive changes can be made instantly at any level

## Key Principles
1. **Keep it simple** — the system must be easy for every player to remember under game pressure.
2. **Tens digit = defense type**: the first digit tells players what kind of defense to play.
3. **Ones digit = depth of pressure**: the second digit tells players where to start the defense.
4. **Verbal cues add detail** without additional numbers — one word can set the tone for how aggressively to guard the inbounds passer.
5. **Consistency across situations** — the same call works after a made basket, a free throw, a timeout, or a turnover.

## The Numbering System

### Defense Type (tens digit)
| Code | Defense |
|------|---------|
| 10 | Combination or gimmick defense (disrupts opponent's set offense) |
| 20 | Normal man-to-man / basic defense |
| 30 | Run-and-jump or run-and-switching defense (ball in front of you) |
| 40 | Trapping and double-teaming (from behind the ball) |
| 50 | Zone defenses |

### Court Depth (ones digit)
| Code | Pressure Starts |
|------|-----------------|
| 4 | Full-court pressure |
| 3 | Three-quarter court pressure |
| 2 | Half-court pressure |
| 1 | Perimeter pressure |

**Example**: "34" = Run-and-jump full-court. "52" = Half-court zone defense. "24" = Full-court man-to-man press.

## Supplementary Verbal Cues
- **"Hard"** — really pressure the inbounds passer; be physical and disruptive.
- **"Soft"** — the inbounds defender drops back into the passing lanes to look to trap or steal the first pass.
- **"Marry" or "Deny"** — deny the opposing point guard or best ball-handler from getting the ball back after passing it. Body-to-body denial pressure.

## Common Mistakes
1. **Overcomplicated system** → Correction: fewer codes that everyone knows beats a sophisticated system that no one remembers under pressure.
2. **No verbal communication on the court** → Correction: the on-court leader (usually PG) echoes the call from the bench loudly before each defensive possession.
3. **Inconsistent application** — calling "30" but some players play man, others zone → Correction: drill the system in practice so each code triggers automatic, rehearsed movement.
