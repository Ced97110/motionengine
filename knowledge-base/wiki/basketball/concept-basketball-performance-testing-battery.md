# Basketball Performance Testing Battery

## Summary
A complete basketball performance testing battery provides athletes, coaches, medical staff, and front office personnel with objective data for decision-making. The NSCA's basketball strength and conditioning text organizes testing into categories covering strength, power, agility, and aerobic capacity — with 15 distinct validated protocols. The overarching goal is not the raw number but the correct interpretation of data in its proper context. [S11, Ch.3]

As sport science and performance testing for basketball continues to evolve, mastery of these foundational protocols is a necessary competency for strength and conditioning professionals at all levels. [S11, Ch.3]

## When to Use
- **Talent identification / recruiting**: Rank and compare athlete physical capabilities objectively.
- **Baseline establishment**: Document starting strength, power, and fitness levels before a training program.
- **Training program evaluation**: Quantify pre-to-post changes from resistance or conditioning programs.
- **Injury risk reduction**: Identify asymmetries, deficits, and faulty movement patterns before they cause injury.
- **Return-to-play progression**: Guide objective decision-making on when athletes are ready to return.

## Complete Test Finder

### Strength Tests
| Test | Primary Measure | Key Equipment |
|---|---|---|
| Isometric mid-thigh pull (IMTP) | Peak force, RFD, impulse | Force plate, fixed bar |
| Hip adduction/abduction | AD:AB strength ratio | Dynamometer |
| Nordic hamstring isometric | Single-leg hamstring strength | Dynamometer |
| T, Y, I isometric shoulder | End-range shoulder girdle strength | Force plate or dynamometer |
| Ballistic push-up (BPU) | Horizontal push force, limb asymmetry | Force plate |
| Bench press max reps | Upper-body muscular endurance | Barbell, rack, spotter |

### Power Tests
| Test | Primary Measure | Key Equipment |
|---|---|---|
| Vertical jump | Lower-body explosive power | Jump mat or Vertec |
| Running vertical jump | Dynamic lower-body power | Jump mat or Vertec |
| Drop jump | Reactive strength index (RSI) | Force plate or contact mat |
| Repeated jump | Jump fatigue, RSI decline | Force plate |
| Rotational medicine ball throw | Rotational power | Medicine ball, tape measure |

### Agility Tests
| Test | Primary Measure | Key Equipment |
|---|---|---|
| Lane agility test | Basketball-specific COD agility | Cones, timing gates |
| Reactive lane shuttle test | Reactive agility | Cones, timing system |
| T-test | Multidirectional agility | Cones, timing gates |
| Three-quarter court sprint | Linear sprint speed | Timing gates |

### Aerobic Tests
| Test | Primary Measure | Key Equipment |
|---|---|---|
| Shuttle run (beep test) | Aerobic capacity (VO₂max estimate) | Cones, audio system |
| VO₂max test (Bruce protocol) | Maximal aerobic capacity | Treadmill, metabolic cart |

## Key Principles
1. **Calibrate equipment every session** — force plate data is only valid with consistent calibration.
2. **Keep a testing log** — longitudinal tracking requires documented protocols and results.
3. **Standardize environment and sequence** — same time of day, same warm-up, same test order.
4. **Maintain test-to-test consistency** — joint angles, body position, equipment placement must be identical.
5. **Interpret contextually** — a 2,534 N IMTP peak force in an NCAA DI man means something different at preseason vs. mid-season vs. returning from injury.

## Common Mistakes
1. **Testing without a plan** → Know before you test what you will do with the data and what thresholds trigger action.
2. **Over-testing in-season** → Testing itself creates fatigue; select the minimum necessary battery and schedule appropriately within the competitive calendar.
3. **Ignoring asymmetries** → Between-limb differences (BPU, hip AD:AB, hamstring) can predict injury before symptoms appear.
4. **Using only large-scale combine data as benchmarks** → Combine data reflects a talent-selected population; use position-matched college data for most athletes.
