# Z-Pattern Run

## Objective
Improve transitional movement and turning ability by sprinting diagonally in a continuous Z-pattern through alternating cone rows.

## Setup
- 1 player
- 6 cones arranged in two parallel rows, 5 yards (4.6 m) apart vertically:
  - Row 1 (top): cones at 0, 10, and 20 yards horizontally
  - Row 2 (bottom): cones at 5, 15, and 25 yards horizontally
- Total horizontal distance: ~25 yards; total run ~70+ yards

## Execution
1. Start in a two-point stance at the start.
2. Sprint diagonally 5 yards (4.6 m) to the closest cone, plant the outside foot, and run around the cone.
3. Sprint diagonally to the next cone and run around it.
4. Continue sprinting diagonally to and around each subsequent cone until all cones are completed.

<!-- DIAGRAM: Six cones in two staggered rows; athlete sprints diagonally between upper and lower rows in a continuous wave/Z-pattern, planting the outside foot to run around each cone. [S8, p.93] -->

## Coaching Points
- Plant the **outside foot** on each cone approach to execute the turn and redirect — this is the key technical cue [S8, p.93]
- Maximize diagonal sprint speed between cones — "attack each diagonal"
- Stay compact around each cone; tight loops are faster than wide ones
- Head swings around to the new focus point to initiate each direction change

## Progressions
1. **Beginner**: Walk-through; practice outside-foot plant at each cone
2. **Intermediate**: Full-speed, timed
3. **Advanced (Complex Variations)**:
   - Start from various positions (lying, four-point stance) [S8, p.93]
   - Change the distance between cone rows and horizontal spacing
   - Use different biomotor skills on alternating legs
   - Cut with inside or outside leg variation
   - Put inside hand on ground during turns
