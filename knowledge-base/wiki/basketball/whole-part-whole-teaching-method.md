# Whole-Part-Whole Teaching Method

## Summary
The whole-part-whole method is described in S6 as one of the best methods to teach basketball skills and techniques. The coach first introduces the complete concept to the entire team, then breaks it into its individual components with breakdown drills, then reassembles the players to run the full action at game speed with increasing defensive resistance. This method applies universally to all footwork, pivoting, and offensive/defensive techniques. [S6, pp.265-266]

> "Your team is only as good as its weakest link." [S6, p.267]

## When to Use
- Introducing any new skill: individual (pivoting, footwork, shot fakes) or team (pin screens, PnR actions, trapping)
- When players cannot yet execute a technique at game speed or under defensive pressure
- In early-season or preseason skills development phases
- Any time a breakdown in technique is observed in a game — return to the whole-part-whole sequence

## Key Principles
1. **Introduce the whole first**: Show every player the complete skill or play with each player's specific responsibilities demonstrated to the entire squad [S6, p.265]
2. **Walk through under scrutiny**: Have every player walk through the technique slowly — players learn best by doing, not just watching [S6, p.265]
3. **Questions phase**: Allow players to ask questions and receive coaching answers before moving to the next phase [S6, p.265]
4. **Speed up progressively**: After the walk-through, gradually increase tempo — slow → medium → game speed [S6, p.265]
5. **Breakdown drills in parts**: Use specific breakdown drills that isolate individual aspects of the technique (e.g., just the cutter footwork, then the screener footwork, then both together) [S6, p.265]
6. **Increase defensive resistance gradually**: Start with dummy/passive defenders, then progressively increase effort as offensive proficiency improves [S6, pp.265-266]
7. **Return to the whole**: Reassemble individuals back into the team setting to run the full skill at game speed [S6, p.265]
8. **Competition requirement**: Final drills must include competition (vs. other individuals or groups) and must be executed at game speed — "Every drill must integrate maximum game-realistic conditions" [S6, p.265]

## Application to Breakdown Drills
- Minimum structure: 1 ball handler + 1 screener + 1 cutter/post + dummy defender on ball handler and pass receiver
- Progress: add dummy defender on the screener once the initial group achieves proficiency
- Rotate all players through all positions (offense AND defense)
- Use short periods (e.g., 55 seconds per station) for efficiency; rotate in 5-second windows
- Entire 12-player team can work in three 4-man groups simultaneously (see [[drill-mass-footwork-rotation]]) [S6, p.266]

## Common Mistakes
1. **Skipping the walk-through** → Players attempt game speed before mechanics are set — correction: always start slow, no exceptions for experienced players [S6, p.265]
2. **Keeping defenders passive too long** → Offensive players never face realistic pressure — correction: escalate defensive effort as soon as proficiency is shown [S6, pp.265-266]
3. **Never returning to the whole** → Players master parts but cannot execute in a team context — correction: always complete the cycle back to team game-speed repetition [S6, p.265]
4. **Not rotating players** → Some players only practice one role — correction: every player must play every position in the breakdown drill [S6, p.266]
