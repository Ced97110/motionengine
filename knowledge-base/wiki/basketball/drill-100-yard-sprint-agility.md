# 100-Yard Sprint (Agility Line Drill)

## Objective
Improve ability to change direction, footwork mechanics, and reaction time through a four-level sprint-touch-return pattern covering 100 total yards.

## Setup
- 1+ players
- Five parallel lines: starting line, lines at 5, 10, 15, and 20 yards from start
- Use court markings or mark with tape/cones

## Execution
1. Start in a two-point stance on the starting line.
2. Sprint 5 yards (4.6 m) to the first line, touch it with the right hand, return to starting line, touch with the left hand.
3. Sprint 10 yards (9 m) to the second line, touch it with the right hand, return to starting line, touch with the left hand.
4. Sprint 15 yards (14 m) to the third line, touch it with the right hand, return to starting line, touch with the left hand.
5. Sprint 20 yards (18 m) to the fourth line, touch it with the right hand, return to starting line, touch with the left hand.

<!-- DIAGRAM: Multi-lane pattern with four target lines at 5, 10, 15, 20 yards from start; athlete sprints out-and-back to each line with alternating right/left hand touches. Eight legs total. [S8, p.87] -->

## Coaching Points
- Alternate right-hand / left-hand touches to ensure bilateral training
- Drop the hips on every turn — no standing upright during direction changes
- Arm drive is critical on the sprint-out and the return
- This is a high-volume drill — use adequate recovery between reps (2:1 rest:work minimum)
- Time and chart each attempt

## Progressions
1. **Beginner**: Complete only the 5-yard and 10-yard legs; rest; add 15-yard leg when conditioned
2. **Intermediate**: Complete all four legs at game tempo
3. **Advanced**: Compete against teammates; add biomotor skill combinations on return legs
