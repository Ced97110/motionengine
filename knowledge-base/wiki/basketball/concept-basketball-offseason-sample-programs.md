# Basketball Off-Season Sample Programs (Guards & Bigs)

## Summary
The NSCA Strength Training for Basketball provides complete 12-14 week off-season sample programs differentiated by position group (Guards vs. Bigs) across three progressive microcycles. Each microcycle contains three training days organized by focus: Lower Body, Upper Body, and Velocity-Based. Programs include exercise order (a/b/c groupings performed as circuits), sets, load (%1RM or bodyweight), reps, tempo (eccentric:pause:concentric), and rest periods [S11, pp.307-320].

The programs are designed to be run in the order prescribed: perform all exercises in a group (1a, 1b, 1c) for one set each before repeating the group for the next set. If exercises have fewer sets than others in the group, perform those sets on the back end of the grouping [S11, p.305].

## Tempo Notation
- Format: `eccentric : pause : concentric` (in seconds)
- Example: `3:1:1` = 3 sec lower, 1 sec pause, 1 sec stand
- `X` = explode during that phase
- `X:X:X` = power exercise with no prescriptive time allotment [S11, p.306]

## PAP (Post-Activation Potentiation) Pairings
The power phase uses PAP pairings — a heavy compound lift followed immediately by an explosive bodyweight movement:
- Back squat (heavy) → Barbell jump squat (BW or AEL) [S11, p.317]
- Bench press (heavy) → Explosive push-up (hands leave ground) [S11, p.317]
- Deadlift (heavy) → Box jump [S11, p.319]
- Seated row (heavy) → Med ball slam [S11, p.319]

The PAP effect: heavy loading temporarily potentiates the neuromuscular system, allowing the subsequent explosive movement to be performed with greater force output.

## Programming Notes
- **Advanced complex** (college/pro): additional exercise groupings (4a, 4b) added at the end of each day for more experienced athletes
- **Stamina focus** notation: certain exercises have a stamina focus (higher reps, moderate tempo) to build local muscular endurance alongside strength/power
- **Velocity targets**: power clean at 1.3-1.4 m/s peak velocity; deadlift at 0.75-1.0 m/s average velocity; back squat at 0.5-0.75 m/s average velocity — velocity-based training (VBT) ensures intent to move fast is maintained
