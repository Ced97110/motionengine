# SAQ Program Design — 6-Week Framework

## Summary
*Training for Speed, Agility, and Quickness* (S8) provides a systematic framework for building 6-week SAQ programs tailored to any sport. The approach begins with a needs analysis — identifying which physical qualities (Speed, Agility, Balance, Quickness, Reaction Time) are most critical for the sport — and then selecting drills by level and emphasis across a 6-week progressive structure. Sample programs for basketball, soccer/lacrosse, and other sports demonstrate how to fill the template. [S8, pp.234-235]

The program template separates needs into rows (Speed / Agility / Quickness for the first 3 weeks; Speed+Quickness / Agility+Quickness / Speed+Agility+Quickness for weeks 4-6) and tracks specific drill names and numbers across a 6-week progression. This allows progressive overload as athletes move from isolated qualities to combined-quality drills.

## When to Use
- Pre-season: build foundational SAQ work from Basic level drills
- In-season: maintain with Intermediate drills; reduce volume
- Off-season: peak training with Advanced drills at full volume
- Remediation: target a weak physical quality (e.g., lateral quickness for slow defenders) with focused 6-week cycle

## Key Principles
1. **Needs analysis first**: Identify the 2–3 most important physical qualities for your sport and position. Basketball = first-step quickness, lateral agility, reaction time, vertical power
2. **Drill progression**: Weeks 1–3 use single-quality drills at Basic/Intermediate; weeks 4–6 combine qualities and add complexity
3. **Volume control**: Start with fewer reps/sets; increase by 10–15% per week maximum (GAS supercompensation principle)
4. **Level matching**: Never assign Advanced drills to athletes who haven't mastered Intermediate; injury risk rises sharply
5. **Variation is key**: Each week advances the drill slightly (e.g., Drill 12 → Drill 13 → Drill 13 alternating fast legs) to prevent neural adaptation
6. **Comments section**: Record athlete performance, pain/soreness, and modifications each week to inform future cycles

## Soccer/Lacrosse 6-Week Sample (Reference Example)
Goals: Improve lateral agility and quickness; train level changes during slide tackles; increase open field acceleration.

## Basketball-Specific Program Design Guidance
Based on basketball's demands (Taylor 2003/2004 — cited in S8 references), prioritize:
- **Speed**: Falling Starts (#25), Gears (#28), Ins and Outs (#29), Ladder Speed Run (#10)
- **Agility**: 20-Yard Shuttle (#53), Carioca (#51), Icky Shuffle (#82), Z-Pattern Cuts (#69)
- **Quickness**: Lateral Skaters (#137), Ball Drops With a Partner (#117), Quick Feet in All Directions (#148), Half Ladder Skill to Sport-Specific Skill (#152)
- **Reaction Time**: Containing-Opponent Drill (#157), Directional Mirror Drill (#167), Icky Shuffle With Reaction (#191)

## The 6-Week Template Structure
```
Weeks 1–3 rows: Speed | Agility | Quickness
Weeks 4–6 rows: Speed+Agility+Quickness | Speed+Quickness | Agility+Quickness
Each cell: Drill Name | Drill #
Comments section at bottom of each week
```

## Common Mistakes
1. **Skipping needs analysis** → Athletes work on wrong qualities; no transfer to sport performance
2. **Using Advanced drills too early** → Injury, poor mechanics, no learning effect
3. **Same drills every week** → Neural adaptation, no progression, boredom
4. **Ignoring reaction time** → Athletes are fast in straight lines but slow to read game situations
5. **No periodization** → Volume stays constant; no supercompensation peak before competition
