# Forward-Oriented Attack

## Overview
A half-court offensive set designed for teams whose best and most versatile scorers are forwards (wing players). Uses hand-offs, basket cuts off the center's screen, post-ups, split action, and staggered double screens to create perimeter, mid-range, and three-point opportunities for the forwards [S13, pp.111-113].

## Formation
- O1 and O2 above the three-point line, ~20 feet apart
- O3 (strong-side forward) and O4 (weak-side forward) at free-throw line extended
- O5 at high post

## Phases

### Phase 1: Hand-Off Entry
- O3 gets open above O5, closer to midcourt, to receive the first pass from O1
- After passing, O1 makes a hard two-step drive toward the basket, then immediately cuts behind O3 for a quick outside hand-off return
- O5 moves to the strong-side elbow to prepare a screen for O3

### Phase 2: Center Screen — Basket Cut or Buttonhook
- O3 takes the defender higher toward midcourt to get the defender above O5, then uses O5's screen to rub the defender off
- **Option A (Buttonhook)**: O3 goes beyond and around O5, turns quickly, and ducks back toward O1
- **Option B (Inside Dive Cut)**: O3 cuts directly to the block
- If O3 gets head and shoulders in front of the defender, O1 delivers the pass for a layup or foul

<!-- DIAGRAM: Figure 5.5 — O3 uses O5's elbow screen, basket cut to the block; O1 looks for layup pass; O2 at top of key -->

### Phase 3: Post-Up and Split Action
- If the basket cut pass is not made, O3 immediately posts up strong on the block
- If O3 is open on the post-up, O1 passes to O3 and runs split action with O2, who has rotated to the top of the key

<!-- DIAGRAM: Figure 5.6 — O1 passes to O3 post-up, O1 and O2 run split action around O3 -->

### Phase 4: Staggered Double Screen for Weak-Side Forward
- After setting the stationary screen for O3's dive (when no post-up pass is made), O5 turns and sets a staggered double screen with O2 on weak-side forward O4, who flashes to the top of the key
- O1 looks deep first (O3); if O3 is not open, passes to O4 at the top of the key
- After passing to O4, O1 slides to the corner to space the floor
- O3's next option: run the baseline and use the same staggered double screen (O2 + O5) to get open on the weak side

<!-- DIAGRAM: Figure 5.7 — O5 and O2 set staggered double screen for O4 flashing to top; O3 runs baseline option -->

### Phase 5: Curl or Fade Off Double Screen
- O2 sets the **first screen** just outside the free-throw lane, then clears straight to the far corner
- O5 sets the **second screen** ~6 feet behind O2's screen
- This spacing allows O3 to **curl for a short jumper** or **fade for a three-pointer**

<!-- DIAGRAM: Figure 5.8 — O2 first screen outside lane (clears to corner), O5 second screen 6 ft behind, O3 reads curl/fade -->

## Key Coaching Points
- O3 must take the defender HIGH before using O5's screen — the timing creates the separation
- "Head and shoulders in front" is the passing trigger on the basket cut
- O1 must look deep first after posting O3 — don't skip to the top of the key prematurely
- The staggered double screen depth determines shot type: tight = curl/short jumper; wide = fade/three
- O4 must be ready to shoot immediately on catching at the top of the key
- The forward is featured throughout: every phase either feeds O3 directly or creates space for O3 as the tertiary option

## Counters
- If the hand-off is denied: O1 drives directly
- If O5's screen is switched: O5 ducks into the post
- If O3's post-up is double-teamed: O1/O2 split action creates an outlet
- If the fade three is contested: O3 takes the curl for a higher-percentage mid-range shot
