# Lob (BLOB)

## Overview
A frontcourt baseline out-of-bounds play designed to free an athletic post player (O5) for a lob finish in the lane. Deception is the core principle: O4 fakes an up screen for O5 while O5 fakes outside, then O3 immediately provides a cross screen that O5 uses to cut hard to the open lane. The inbounder (O1) delivers the lob where O5 can make the play.

## Formation
- **O1**: Inbounder on the right side underneath the basket
- **O4**: Left low block
- **O5**: Left elbow (primary cutter)
- **O3**: Right elbow (cross screener)
- **O2**: Right corner (floor spacing)

## Phases

### Phase 1: Deception Setup
- On the command "Go," **O4 pretends to set an up screen for O5**
- **O5 makes a jab fake to the outside** as if going left — drawing the defender that direction

### Phase 2: Cross Screen for O5
- **O3 reacts immediately** with a **cross screen for O5**
- **O5 pivots off the left foot**, using O3's screen
- **O5 cuts hard down the middle** of the lane, looking for the lob

### Phase 3: Lob Delivery
- **O1 reads the defense** and delivers the lob pass to where O5 can make the play (dunk or layup)
- **O2** in the corner maintains floor spacing and keeps the corner defender occupied

## Key Coaching Points
- **Deception execution is paramount** — O5's outside jab fake must be convincing; if O5 simply runs to the screen, the defense reads it immediately
- **Floor balance** is essential: the court must be spread enough that O5 has room to operate in the lane (O2 in the corner fulfills this)
- **O3's cross screen timing** must be immediate after O4's fake screen — any delay allows the defense to recover
- **O1 must read the defense**: the lob should be delivered to the open space where O5 can attack the ball above the rim
- **O5's cut must be hard** — casual cuts in lob plays result in deflections and turnovers
- The inbounder (O1) has 5 seconds; the entire sequence must be fast

## Common Mistakes
1. O5's jab fake too casual → defender doesn't bite, reads the cross screen
2. O3 too slow to react with the cross screen → defense recovers before O5's cut
3. Lob too high or short → O5 cannot make the play
4. Poor floor spacing → lane congested, no room for lob cut
