# Youth Basketball Practice Schedules

## Summary
Burrall and Patrick Paye provide six complete sample practice schedules in the Appendix of *Youth Basketball Drills* — one 30-minute and one 90-minute plan for each of three season phases (early, mid, late). Each schedule lists specific drill numbers, the time allocated to each drill, and the coaching rationale for including it. These templates are designed to be adapted, not copied verbatim — coaches should substitute their own drill numbering while following the time-allocation and sequencing logic [S15, pp.339-343].

The key principle is that practice schedules must answer four questions: (1) What phase of the season is it? (2) How much time do I have? (3) What are we running offensively and defensively? (4) What broke down in the last practice or game? Games are won or lost in practice — the schedule must always push toward continuous improvement.

## When to Use
- Pre-season planning to map fundamentals across the full season calendar
- After a game loss or performance breakdown to prioritize corrective work
- When returning from a layoff (more review + scrimmage)
- When practice time is compressed (30-min version appropriate for youth leagues with limited gym time)

## Key Principles
1. **Match the schedule to the season phase** — Early: heavy fundamentals, introduce offensive/defensive system. Mid: review and troubleshoot. Late: sharpen, maintain, scrimmage [S15, pp.339-343].
2. **Let game performance drive drill selection** — After a turnover-heavy game, increase ball protection and trapping drills. After a shooting slump, increase shooting time [S15, p.342].
3. **Sequence offense before defense, fundamentals before team** — Warm-up → individual skills → team offense → team defense → scrimmage is the standard flow.
4. **Include scrimmage every late-season practice** — Live offense vs. live defense shows whether drill work is transferring to game situations [S15, pp.340, 343].
5. **After a layoff, re-condition before reviewing tactics** — Get back to game fitness first, use more scrimmage time, then taper back to fundamentals over subsequent practices [S15, p.343].

## Common Mistakes
1. **Using the same schedule every practice** → game performance must continuously reshape the plan; diagnose what broke down and target it
2. **Skipping scrimmage in late season** → players need live reps to transfer drill work; always include live offense vs. defense in late season
3. **Ignoring conditioning after a layoff** → start with conditioning drills before review; physical readiness precedes tactical sharpness
4. **Over-scheduling in a 30-minute window** → use the 30-min templates as ceilings, not floors; it is better to do fewer drills well than rush through all of them
5. **No post-game reflection** → the question "What did not work in the last game?" must be answered before building each new practice plan [S15, p.339]
