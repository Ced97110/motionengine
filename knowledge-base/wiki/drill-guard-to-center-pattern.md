# Guard-to-Center Pass Pattern (LA Offense)

## Objective
Teach the guard-to-center entry option of the LA Offense, using the high-post center as a ball-movement hub with an automatic backdoor first option, a guard-around second option, and a double-stack screen third option — plus three pressure releases.

## Setup
- Half court, 5 players
- O1 and O2 at guard positions; O3 at strong-side wing; O4 at weak-side forward; O5 at high post (hands up, feet shoulder-width apart)
- This pattern can begin from midcourt when defensive pressure is applied
- Either guard can initiate

## Execution

### Main Sequence
1. O1 makes a high-post pass to O5 and sets a screen for the opposite guard (O2).
2. When O5 receives the ball, strong-side O3 makes an **automatic backdoor cut** (First Option — timing play, extremely effective against pressure).
3. O5 looks for O3 on the backdoor first.
4. After the backdoor look, O5 checks whether O1's screen frees O2 (Second Option).
5. If O2 is open, O5 passes to O2 for a jump shot or drive — O2 has the entire side of the court for a 1-on-1.
6. If O2 is not open, O3 uses the **double-stack screen** set by O4 and O1 (Third Option) for a catch-and-shoot jump shot.

### Releases
When all three options fail:
1. O5 immediately uses a **dribble handoff** as O3 keeps coming to the ball.
2. O4 dives to the middle off O1, who has the inside lane.
3. O5 passes the ball to O2, who has relocated on the sideline (free-throw line extended), and runs a **pick-and-roll**.

## Coaching Points
- O3's backdoor is **automatic** on every guard-to-center pass — "as soon as O5 catches, O3 cuts."
- O5 must catch, face the defender, and read — don't panic and use a dribble handoff too early.
- O2's guard-around only works if O5 waits for the backdoor look first — sequence matters.
- In the double-stack screen, O4 and O1 must set the stack simultaneously and legally.
- **Safety valve reminder**: After every half-court offensive set, one player rotates back to protect the basket; another contains the ball on transition.

## Progressions
1. **Beginner**: Walk through the backdoor + guard-around sequence only; introduce the double stack separately.
2. **Intermediate**: Practice all three options against a token defense; run releases when called by the coach.
3. **Advanced**: Live 5-on-5 defense; initiate from backcourt pressure to simulate the full pattern.
