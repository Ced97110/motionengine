# Pin Down Pull Up Drill

## Objective
Simulate pin-down screen action while adding a conditioning component (mid-court touch sprint) between shots — builds both catch-and-shoot accuracy under fatigue and game-speed footwork.

## Setup
- Half-court
- One chair (with ball) on the wing spot
- One chair (with ball) in the central lane of the half-court (foul line area)
- 1 rebounder in the lane
- Player starts at the low-post position

## Execution
1. Player fakes a cut inside (simulating a screen-away action)
2. Sprints to the **wing chair**, picks up the ball, squares up, jump shot
3. Sprints to the **mid-court line**, touches it with one foot
4. Sprints back to the **central lane chair**, picks up the ball, squares up, jump shot
5. Run for set reps or set time

## Coaching Points
- Fake inside must be convincing — sell the cut before breaking to the wing
- Pick up the ball **low** off the chair — stay low, rise into the shot
- Mid-court sprint must be full effort — this is the conditioning component, not a jog
- "Perfect feet" on both shots despite the sprint fatigue
- This drill simulates the physical reality that shooters in games are never fully rested

## Progressions
1. **Beginner**: No sprint to mid-court; just alternate between two wing chairs
2. **Intermediate**: Full drill with mid-court touch as described
3. **Advanced**: Add a time goal (e.g., 10 makes in 90 seconds); add a live closeout on the second shot
