# Four Individual Defensive Traits Drill (#144)

## Objective
Develop a complete individual defender by forcing X1 to execute all four major defensive traits — on-ball 1-on-1, deny the wing, help-side positioning, and rotation — in one continuous gauntlet against 5 offensive players.

## Setup
- Half court
- 1 defender (X1) and 5 offensive players (1 at ball-side wing, 2 near top/wing, 3 at strong-side elbow area, 4 at corner, 5 at opposite corner/baseline)
- See diagram: player 1 ball-side low, players 2 and 3 at the elbows, player 4 right wing, player 5 right corner
- Rotation after X1 completes the drill: X1 → 1 → 2 → 3 → 4 → 5 → X1

## Execution

### Phase A — 1-on-1 On Ball
1. Player 1 uses fakes before the dribble; then has only **3 dribbles** to create separation.
2. X1 plays full on-ball defense using all defensive footwork (front-foot-to-pivot-foot, advance/retreat/swing steps).
3. Phase ends when: player 1 shoots, player 1 fails to create separation, or X1 defensively rebounds.
4. Whoever gets the ball passes to player 2. Player 1 and X1 return to original positions.

### Phase B — Deny the Wing / Close the Gap / Rebounding
1. Player 2 has three options (coach selects or player chooses):
   - **Option 1:** Pass to player 1 breaking the sideline → X1 uses **deny-the-wing** fence slide technique.
   - **Option 2:** Dribble drive on X1's side → X1 uses **close-the-gap** techniques.
   - **Option 3:** Shoot → player 1 and X1 fight for the **rebound** using offensive and defensive techniques.
2. Whoever ends with the ball passes to player 3.

### Phase C — Help-Side (Flat Triangle)
1. Player 3 receives the ball. Player 1 and X1 return to original positions. X1 is now on the **help side**.
2. X1 immediately establishes **flat triangle** position using ball-you-man techniques.
3. Player 3 has three options:
   - **Option 1:** Pass to player 4 → X1 adjusts help-side positioning.
   - **Option 2:** Pass back to player 2 → Phase B re-activates.
   - **Option 3:** Shoot → player 1 and X1 fight for the rebound.
4. Whoever gets the rebound passes to player 4.

### Phase D — Rotation and Advanced Skills
1. Player 1 and X1 return to original positions. X1 gets into proper help-side position.
2. Player 4 has four options:
   - **Option 1:** Take a shot → X1 and player 1 fight for the rebound.
   - **Option 2:** Dribble drive a few steps then pass to player 1 → X1 must **close out** on the pass receiver.
   - **Option 3:** Motion for player 1 to flash pivot cut → X1 must **deny the flash pivot cut**.
   - **Option 4:** Pass to player 5, who drives the baseline → X1 must **rotate and draw the charge**.

## Coaching Points
- Your team defense is only as good as your weakest defender — make sure every player executes correctly. [S15, p.302]
- **Early in the season:** Stop the drill as mistakes occur and correct immediately.
- **Later in the season:** Let the four phases run continuously; then discuss mistakes with the defender afterward.
- Coach controls which options are available in each phase — tailor the drill to current defensive priorities.
- Each phase can also be used as a standalone drill (e.g., using only Phase D activates: rebounding, closeout, flash-pivot denial, and 1-on-1 footwork all at once). [S15, p.302]
- "Make sure the defender is executing properly. This depends on which option in which phase you are using." [S15, p.302]

## Progressions
1. **Beginner**: Player chooses one option from phases A, B, C only. Player does not tell the defender which option is chosen — defender must read and react.
2. **Intermediate**: Players must choose one option from EACH phase — forces the defender to execute 4 continuous defensive plays consecutively.
3. **Advanced**: Players 2, 3, 4, and 5 each independently choose which options they intend to employ — requires the defender to execute multiple techniques within each phase.

**Mixing options example:** Player 1 and X1 play 1-on-1 (Phase A), player 2 compels X1 to deny wing (Phase B), player 3 shoots (Phase C), player 4 forces X1 to hedge-and-recover before player 5 drives baseline (Phase D). [S15, p.302]
