# Vertical Jump and Running Vertical Jump Testing

## Summary
The vertical jump test (stationary countermovement) and running vertical jump test (dynamic approach) are the two primary power tests for basketball athletes. Both assess sagittal plane vertical power but differ in their specificity to game actions. The stationary vertical jump correlates with rebounding from a set position; the running vertical jump (also called the maximum vertical leap in the NBA) correlates with fast-break finishes, dunks, and dynamic play. Both tests have been shown to predict basketball talent level [S11].

Normative data are available at all three competitive levels (high school, NCAA, NBA Combine), making these the most benchmarked tests in basketball S&C.

## Vertical Jump Test

### Protocol
1. Establish standing reach height (dominant hand, flat-footed)
2. Set Vertec to a challenging but reachable height
3. Position athlete directly under the Vertec
4. Athlete performs a maximum-effort CMJ — **no step allowed, countermovement only**
5. Record: peak Vertec touch minus standing reach = vertical jump height
6. ≥2 min rest between trials; use best score

**Equipment options:** Vertec stand (most common), contact mat/photocell mat, motion capture

## Running Vertical Jump Test

### Protocol
1. Establish standing reach height (dominant hand, flat-footed)
2. Mark off a 15-foot (4.6 m) arc boundary around the Vertec (e.g., free-throw line extended to baseline)
3. Set Vertec to a challenging height
4. Athlete takes as many steps within the 15-ft arc as needed to gain momentum
5. Self-select single- or double-leg takeoff (unless specified)
6. Record: peak touch minus standing reach = running vertical jump height
7. ≥2 min rest between trials; use best score

**Key distinction from stationary VJ:** Momentum from the approach allows greater height, revealing dynamic power capabilities beyond the stationary test [S11]

## When to Use
- **Stationary VJ**: assess pure vertical power from a stopped position; relevant to half-court rebounding and shot-blocking
- **Running VJ**: assess dynamic power in transition; relevant to fast-break finishes, dunks, and rim attacks off cuts
- Use **both tests together** — the gap between the two reveals how well an athlete converts dynamic momentum into vertical height

## Key Principles
1. **Standing reach is the baseline** — always measure it first and record it precisely
2. **No step on the stationary jump** — countermovement only; a step invalidates the test
3. **Enforce the 15-ft arc** on the running test — standardize approach distance across athletes
4. **Allow ≥2 min rest between trials** — fatigue suppresses jump height significantly
5. **Report the single best trial** — do not average multiple attempts

## Common Mistakes
1. Not establishing or recording standing reach → can't calculate jump height
2. Letting athletes take a step on the stationary VJ → inflates scores; invalidates comparison to norms
3. Inconsistent approach distances on the running VJ → compare only within standardized conditions
4. Comparing female scores to male normative tables → always use sex-appropriate data
