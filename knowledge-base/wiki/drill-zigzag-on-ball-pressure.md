# Zigzag On-the-Ball Pressure Drill

## Objective
Build on-the-ball defensive footwork — step-and-slide to cut off the lane and sideline — and the wrist-crossed jamming technique when the dribbler picks up the ball.

## Setup
- 2 players per group (1 offense, 1 defense)
- Players start in one corner, outside the baseline
- Half-court or full court
- No additional equipment needed
- Can run simultaneously at both ends

## Execution

### Phase 1: Guided Zigzag (Lane–Sideline)
1. Offensive player dribbles hard toward the lane.
2. Defender steps and slides to cut off the path to the lane.
3. Offensive player reverses and dribbles hard to the sideline.
4. Defender steps and slides to cut off the sideline.
5. Continual back-and-forth (lane → sideline) from baseline to half-court.
6. At half-court, players exchange roles and return to baseline the same way.

### Phase 2: Same as Phase 1, with Ball-Jam
1. Same zigzag as Phase 1.
2. At any point, the dribbler takes two hard dribbles and picks up the ball.
3. Defender immediately raises both hands with **wrists crossed** to jam and mirror the ball, preventing a pass.
4. Dribbler picks up and raises the ball 3–4 times; defender jams each time.
5. Exchange roles; repeat.

### Phase 3: Ball-Jam Only (Short Version)
- Start the same as Phase 2 but the emphasis is entirely on the jamming technique.
- Dribbler picks up the ball immediately; defender jams 3–4 times per repetition.
- Very short, intense; exchange roles after each jam cluster.

### Phase 4: Live 1-on-1 to Half-Court
1. Offensive player plays normally and tries to beat the defender to half-court.
2. Defender uses step-and-slide to cut off all paths, as in a game situation.
3. At half-court, roles are exchanged.

## Coaching Points
- **"Step, don't reach"** — body position cuts off the lane; hands stay up.
- Crossed-wrist jam: elbows bent and close, wrists crossed, hands at ball height; this positioning allows quickest deflection.
- Defender must never concede a direct path to the basket on the first dribble.
- Phase 4 is live; if the dribbler blows by, the defender sprints to back-tip — never quit.

## Progressions
1. **Beginner**: Phase 1 only — passive dribbler, exaggerated slides.
2. **Intermediate**: Phases 1–3 combined — forced pick-up and jam practiced multiple times.
3. **Advanced**: Phase 4 live, add a second defender at half-court for a handoff/continuation drill.
