# Mass Wave Sliding Drill (#145)

## Objective
Develop correct on-ball defensive footwork responses in all players simultaneously through coach-led mass repetitions of advance, retreat, and swing steps.

## Setup
- Full team in the frontcourt, staggered positions so all players can see the coach
- Coach at the bottom of the key area, players spread in a wave formation
- No ball needed (coach simulates offensive moves)
- 1 minute duration

<!-- DIAGRAM: Full team spread across frontcourt in 3-4 staggered rows, all facing coach who stands near the low block/paint. Players react as a wave. -->

## Execution
1. Coach establishes a pivot foot. Change from day to day.
2. All defensive players adopt the proper stance: front-foot-to-pivot-foot, front-foot-to-free-foot, or parallel — based on the designated pivot foot.
3. Coach **jab-steps** → defenders react with a **retreat step**.
4. Coach **pulls back from the jab step** → defenders react with an **advance step**.
5. Coach does a **jab-step crossover** → defenders react with a **swing step**.
6. Coach uses a **spin move** → defenders react with a quick one-two step, re-establishing a new front-foot-to-pivot-foot, front-foot-to-free-foot, or parallel stance.
7. Coach **begins a dribble** in one direction → players react by **overplaying the dribble**.
8. Coach **changes direction** while dribbling → players react with a retreat step with the trail foot and quick movement into another overplay.
9. Coach **picks up the dribble** (both hands on ball) → defenders react with an **advance step** and **tight overplay**.
10. Coach may have players gather in the frontcourt and repeat the full sequence.

### Advanced Option (with Assistant Coach)
- Tell players they are on the weak side.
- Assistant coach stands parallel to the head coach. Players react by **jumping to the ball** (fence-slide direction) on the first pass.
- Assistant moves downcourt → players react by **denying** with fence-slide position.
- Assistant moves to the weak side → coach throws a **skip pass** → players **close out** to the ball.
- Coach throws a **line-drive pass** → defenders execute **interception stance** and intercept the pass. [S15, p.305]

## Coaching Points
- Change the designated pivot foot daily so defenders get equal reps on both sides.
- All reactions must be simultaneous — the whole team should move as one wave.
- Teach players the exact vocabulary: advance step, retreat step, swing step, overplay, interception stance.
- This is a warm-up and review tool — 1 minute is sufficient for daily use.
- Pairs perfectly as a lead-in to any live 1-on-1 defensive drill.

## Progressions
1. **Beginner**: Basic sequence (steps 1-10 above) without an assistant coach.
2. **Advanced**: Add the two-coach skip-pass and interception-stance option to simulate ball movement from both sides of the court.
