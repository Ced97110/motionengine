# Inside Pick-the-Picker (BLOB)

## Overview
A frontcourt baseline out-of-bounds play designed to produce a score off the initial pass. Normally run as a double-stagger play for a corner shooter (O2), it becomes most effective as a deception play: O2 acts as the decoy while O5 (the first screener) attacks the basket for a layup off a back screen by O4. Used famously with the Chicago Bulls during the Michael Jordan era — reportedly good for at least one basket per game.

## Formation
- **O3**: Inbounder on the baseline
- **O4**: Strong-side block (right of inbounder)
- **O2**: Weak-side block
- **O5**: Just below the free-throw line (screener / primary cutter in the counter)
- **O1**: Top of the key (safety / ball-handler)

## Phases

### Phase 1: Basic Action (Decoy — Run First)
*Run this version 2-3 times before using the counter.*
- **O5 sets a screen for O2** to come off
- O2 cuts toward the corner looking for a catch-and-shoot jump shot off the double-stagger (O5 then O4)
- O3 delivers the pass to O2 in the corner for a jump shot

### Phase 2: Counter Action — Inside Pick-the-Picker
*After establishing the basic version, run the counter.*
- The play call changes; **O2 becomes the decoy**
- **O5 starts the same screen action for O2**, drawing defensive attention
- **O2 waits** — does NOT cut immediately — and only cuts when **O5 gets lower on the block**
- **O5 is now on the low block opposite O3** (vacated space)
- **O4 sets a solid back screen for O5**
- **O5 cuts hard to the vacated open space** (the lane) for a layup
- O3 delivers a direct pass to O5 for the layup

## Key Coaching Points
- **Deception is essential** — must run the basic (O2 corner) version multiple times before the counter works
- **O2's patience** is the key: O2 must NOT cut early; wait until O5 drops to the block
- O4's back screen must be solid and on time — set it as soon as O5 drops
- O5 must cut *hard* to the open space — no hesitation
- O3 reads O5's cut and delivers a direct pass; do not bobble or delay
- Works best when O2 is a credible threat from the corner — the defense must honor the decoy

## Counters
The entire play IS the counter to the basic action. If the defense adapts:
- If X5 anticipates O5's cut, O3 can lob to O4 who holds his screen position
- O1 at the top of the key serves as a safety if neither option is available
