# Basketball Preseason Sample Microcycle — Bigs (Power Phase, Weeks 11-14)

## Summary
This four-day microcycle represents a power-emphasis off-season block (Microcycle 3, Weeks 11–14) for post players (bigs: PF and C). Days 1 and 2 use a standard PAP/power approach; Days 3 and 4 add velocity-based training to the lower and upper body sessions respectively. The program uses cluster/PAP pairings (indicated by letter groupings: 1a, 1b, 1c), velocity-based primary exercises, and position-based advanced complexes for college/pro athletes. [S11, Table 10.6 and p.3]

**Key to reading tables:**
- **Order:** Perform one set of each exercise in the group (1a, 1b, 1c), then cycle back for the next set
- **Tempo:** Eccentric : Isometric hold : Concentric (seconds); "X" = explode
- **X:X:X** = power exercise, no prescribed time per phase
- **2:1:X** = 2 s eccentric, 1 s hold at bottom, explosive concentric

## Key Programming Notes
- The "cluster" back squat (set 5 with *) uses 40–50% load with 2 reps × 3–4 intrasets with 20 s intraset rest — the PAP effect from the heavier sets potentiates these lighter explosive reps
- The PAP pairing of heavy squat/bench with jump squat/med ball throw is the heart of Days 1 and 2
- Day 3's isometric mid-thigh pull at 100% effort (sets 3–5) is a maximal effort force production test/training tool that potentiates the box jump (1b)
- Advanced complex (4a, 4b, 4c) is designated for college or pro athletes only — requires greater technical proficiency
