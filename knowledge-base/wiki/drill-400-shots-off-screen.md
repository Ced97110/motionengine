# 400+ Shots Off a Screen (Comprehensive Screen Workout)

## Objective
Build complete off-screen scoring mastery by combining all screen reads — fade, fill, curl, dribble fade, dribble fill, and dribble curl — into a single high-volume training session that bridges shooting off screens with live-ball attacking moves.

## Setup
- **Players**: Shooter (cutter), passer, stationary screener (player or cone/chair)
- **Court area**: Half court — wing, elbow, paint, baseline
- **Passer position**: Top of the key
- **Shooter start position**: Underneath the basket
- **Screener position**: Wing or elbow (stationary)
- **Equipment**: Multiple basketballs for pace; shot tracking log

## Execution
1. Shooter begins underneath the basket and executes the V-cut to set up the screen
2. Work through all six shot types in sequence (or as designated by coach):
   - **Fade** — catch on wing, shoot off the screen (no dribble)
   - **Fill** — catch on wing at 15–20 ft, catch-and-shoot
   - **Dribble Fade** — catch on wing, read high closeout, attack baseline with dribble
   - **Dribble Fill** — catch on wing, read low closeout, attack middle with dribble
   - **Curl** — curl to elbow (10–15 ft), catch-and-shoot
   - **Dribble Curl** — curl to elbow, defender behind, continue attacking the paint with dribble
3. Shoot each type in sets of 5–10 reps, alternating with 5 free throws between sets
4. Work both left and right sides of the floor for each shot type
5. Total session volume: 400+ shots (field goal attempts + free throws combined)

## Coaching Points
- This is the master workout — all previous workouts (#5-5 through #5-18) are building blocks for this session
- "Read, react, and score" — every rep should begin with a defensive read before the catch
- Demand game speed: V-cuts must be sharp, catches must be decisive, and attacks must be committed
- Track makes and misses by shot type to identify the weakest read
- The passer must time throws to the shooter's rhythm — the pass creates the rhythm, the rhythm creates the shot
- Include free throws to simulate the game demand for FT shooting under fatigue
- "Any scoring opportunity" — reinforce the mindset that the dribble attack opens multiple options: shoot, floater, layup, or pass out to open teammate

## Progressions
1. **Structured**: Run each of the six drill types from Workouts #5-5 through #5-18 back-to-back in a single session
2. **Random**: Coach calls the shot type after the shooter catches the ball (read training)
3. **Live Defender**: Add a live defender who actually guards the cutter — the read becomes real, not simulated
